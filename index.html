SSOO AUTOMÁTICO DE CULTIVOS

SOBRE NUESTRO SISTEMA OPERATIVO, SOLO PARA EL PROFE Y COMPAÑEROS

Este sistema consta de una automatización para poder realizar un proceso continuo y completo de distribución de alimentos cultivados en una parcela. Va a abarcar los siguientes pasos, desde el cultivo de alimentos hasya el empaque de los mismos:

1. Sistema de recolección de cultivos.

2. Sistema de clasificación de cultivos.

3. Sistema de preparación de alimentos.

4. Sistema de almacenamiento.

5. Sistema de empaquetamiento.
